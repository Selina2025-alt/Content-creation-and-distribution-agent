页面 2 · 两种安装入口
入口 A · 上传 zip 技能包
拖拽 zip 到这里，或点击上传
要求：压缩包根目录必须包含 SKILL.md
选择 zip 文件
系统自动解压并校验：是否存在 SKILL.md
读取 SKILL.md、必要的 references，提炼规则和适用平台
生成本地摘要和索引，加入 Skills 库
入口 B · 一句话仓库安装
从 GitHub 安装 Skill
请帮我安装 openai/skills 仓库里的 xxx 技能
或
请帮我安装 owner/repo 中 path/to/skill 这个技能
解析仓库与路径
下载 skill 目录
安装到本地 skill 库
第一阶段只支持 GitHub 仓库来源。成功安装后，skill 会和 zip 上传的 skill 一样进入本地资源库。
